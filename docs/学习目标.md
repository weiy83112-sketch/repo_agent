# Mission: AI Agent Development

## Why

I want to build enough AI Agent engineering ability to compete for Agent development roles, not just Agent testing roles. The learning should produce both real project code and a durable knowledge system I can review before interviews.

## Success looks like

- I can explain core Agent concepts in Chinese while recognizing the English terms used in open-source projects and documentation.
- I can build a Python Agent project with LangGraph, LangChain, and DeepSeek model routing.
- I can describe the project architecture in interviews: state machine, tool calling, model router, memory, RAG, evaluation, and deployment tradeoffs.
- I maintain HTML lessons and reference pages while building, so knowledge becomes a connected system rather than scattered notes.

## Constraints

- Use Chinese as the main teaching language and keep key English terms beside the Chinese names.
- Prefer short, focused lessons that connect directly to the Agent project.
- Maintain knowledge in HTML files under the teach workspace.
- Avoid adding unnecessary local model complexity unless a future company requirement makes it relevant.

## Out of scope

- Local model deployment as a first-class goal.
- Deep theoretical machine learning.
- Building a polished frontend before the Agent core is understood.
