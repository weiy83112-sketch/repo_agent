这张图用于维护脑中的 Agent 体系。每学一个知识点，就把它放回这张图里。

下一批建议学习

  1. 手写一个最小 Agent 循环：把 messages、工具调用、条件边串起来。
  2. 模型路由（Model Router）：如何让 Agent 面向能力，而不是写死模型。
  3. RAG：Agent 如何查自己的知识库。
  4. Checkpoint：长流程 Agent 如何恢复和追踪。

维护规则

  • 每个新概念先进入术语表，再决定是否做成单独课程。
  • 每节课只解决一个概念，避免一口气塞太多。
  • 学完后写一条 learning record，记录真正理解了什么。
  • 知识图谱只放结构，不放长解释；长解释放 lesson 或 glossary。
