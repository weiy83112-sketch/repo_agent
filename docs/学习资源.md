# AI Agent Development Resources

## Knowledge

- [LangGraph Quickstart](https://docs.langchain.com/oss/python/langgraph/quickstart)
  Use for: the practical shape of LangGraph agents, including state, nodes, tool calling, and conditional flow.
- [LangGraph Concepts](https://docs.langchain.com/oss/python/langgraph/concepts)
  Use for: deeper terms such as state, graph, durable execution, memory, and human-in-the-loop.
- [LangChain Python Documentation](https://docs.langchain.com/oss/python/langchain/overview)
  Use for: model integration, messages, tools, retrieval, and provider abstractions around LLM applications.
- [LangChain Messages Documentation](https://docs.langchain.com/oss/python/langchain/messages)
  Use for: understanding HumanMessage, AIMessage, ToolMessage, and how chat models consume message lists.
- [LangChain Tools Documentation](https://docs.langchain.com/oss/python/langchain/tools)
  Use for: understanding tools as callable units with schemas, names, descriptions, and inputs.
- [LangChain Structured Output Documentation](https://docs.langchain.com/oss/python/langchain/structured-output)
  Use for: understanding how model output can be constrained into program-readable fields.
- [DeepSeek API Documentation](https://api-docs.deepseek.com/)
  Use for: DeepSeek model names, API compatibility, reasoning settings, and production model configuration.
- [OpenHands GitHub Repository](https://github.com/All-Hands-AI/OpenHands)
  Use for: studying a real large-scale coding Agent system after the core concepts are familiar.
- [Chrome DevTools Documentation](https://developer.chrome.com/docs/devtools)
  Use for: browser debugging, Elements, Console, Network, Sources, and frontend inspection workflows.
- [MDN Same-origin policy](https://developer.mozilla.org/en-US/docs/Web/Security/Same-origin_policy)
  Use for: understanding browser security boundaries, including why local files and web origins behave differently.
